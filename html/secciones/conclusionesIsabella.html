Mi Experiencia en el Proyecto

Durante el desarrollo del proyecto Doggie, amplié mis conocimientos sobre las distintas funcionalidades de HTML, CSS y JavaScript. A medida que iba desarrollando la web, pude profundizar en estos lenguajes y comprender mejor su aplicación. Por otro lado, mejoré significativamente mis habilidades en la gestión de repositorios, especialmente al hacer commits y push. Estas acciones solían llevarme mucho tiempo, pero con la práctica adquirida en este proyecto, ahora las realizo con mayor fluidez y seguridad.

Aprendizajes Clave

  • Uno de los principales aprendizajes fue la importancia de estructurar bien el código y mantener una organización clara en los archivos.
  • También reforcé la necesidad de realizar commits frecuentes y con mensajes descriptivos, lo que facilita el seguimiento del proyecto y la colaboración en equipo.
  • Además, comprendí mejor cómo optimizar el uso de CSS y JavaScript para mejorar la interactividad de la web.

Resultados Obtenidos

El principal resultado del proyecto fue el desarrollo exitoso de una página web funcional y bien estructurada. A lo largo del proceso, logré aplicar y mejorar mis conocimientos en HTML, CSS y JavaScript para crear una interfaz atractiva y dinámica. Implementé un diseño responsivo y bien organizado, asegurando que la web se adaptara a diferentes dispositivos. Además, gracias a JavaScript, añadí funcionalidades dinámicas que enriquecieron la experiencia del usuario, haciendo que la navegación fuera más interactiva e intuitiva.

Próximos Pasos

Para futuras iteraciones del proyecto, recomendaría implementar un sistema de reservas en línea, integración con redes sociales para compartir contenido y una sección de blog para compartir consejos sobre el cuidado de mascotas.
